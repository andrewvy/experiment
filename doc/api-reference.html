    Modules

    Experiment

    This module injects the lab config for working with labs. This also has the methods for creating and running lab experiments

    Experiment.Adapters.Default

    This is the default lab adapter for Experiment. Records the output to Logger.info

    Experiment.Base

    This is the base behaviour for creating a Lab Adapter for recording the results of experiments

    Experiment.Lab

    This module defines the Experiment.Lab struct and the main functions for working with the lab. This struct holds all of the experimental functions, as well as the control functions

    Experiment.Utils

    This module holds utility functions needed by Experiment
